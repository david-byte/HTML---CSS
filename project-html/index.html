HTML

HTML significa Hyper Text Markup Language (Linguagem de Marcação de Hipertexto). O HTML descreve a estrutura da página Web através de elementos que serão informados ao navegador como exibir o conteúdo.

Estes elementos são chamados de "tags", elas rotulam o conteúdo, como: "cabeçalho", "parágrafo", "tabela" e assim por diante. Os navegadores não exibem as tags HTML, mas usam para sua renderização.

Estrutura do HTML

A estrutura do HTML é bem simples, temos: html, head, e body. Para utilização do HTML5 temos a tag inicial que é o !doctype html, assim o navegador entendera que seu código está em HTML5, logo após vem a estrutura acima citade de html head e body.

Atributos

Todos os elementos podem ter atributos, os atributos fornecem informçaões adicionais de um elemento.

Atributo href

Os links HTML são definidos com a tag "a". O endereço do link é especificado no href.

Atributo src

O nome do arquivo fonte da imagem é especificado no src

Atributo width e height

As imagens HTML também têm width e height como atributos, especificam largura e altura.

Atributo alt

O alt especifica um texto alternativo a ser usado caso a imagem não seja exibida. O valor alt também pode ser ligo por leitores de tela.

Atributo style

O atributo style é usado para especificar o estilo de um elemento como: cor, fonte, tamanho etc.

Atributo lang

Este atributo serve para mostrar ao navegador o idioma e também é importante para mecanismos de acessibilidade (leitores de tela).

Atributo title

O atributo é adicionado ao tag de parágrafo e seu valor será exibido como uma dica de ferramentaquando você passa o mouse sobre.

Cabeçalhos

Os cabeçalhos são definidos do h1 até o h6 em ordem de importância, sendo o h1 o mais importante. Os mecanismos de pesquisas usam os títulos para indexar a estrutura e o conteúdo de suas páginas web.

Elementos de formatação

Os elementos de formatação foram projetados para exibir tipos especiais de texto:

b - texto em negrito
strong - texto importante
i - texto em itálico
em - texto enfatizado
mark - texto marcado
small - texto pequeno
del - texto excluído
ins - texto inserido
sub - texto subescrito
sup - texto sobrescrito
